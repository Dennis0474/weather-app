# 🌤️ Dynamic Weather App

A dynamic weather app that fetches real-time weather data and displays
it with matching themes, icons, and animated GIFs.

## Features
- 🌍 Search weather by any location
- 🌡️ Toggle between °C and °F
- 🎨 Dynamic themes based on weather condition
- 🎞️ Animated GIFs via Giphy API
- 💾 Saves last search and unit preference via localStorage

## Tech Stack
- Vanilla JavaScript (ES6 Modules)
- Visual Crossing Weather API
- Giphy API
- HTML5 & CSS3

## Project Structure
- weather-app/
- weather.html
- weather.css
- main.js
- weatherService.js
- gifService.js
- themeService.js
- storage.js
